# Get-Markdown

Gets text that represents a markdown document for the specified PowerShell help object.

```Get-Markdown [-HelpObject] <PSObject>```

## Parameters

### ```-HelpObject```

*Custom object that represents command help.*

  <tr><td>Type:</td><td>PSObject</td></tr>
